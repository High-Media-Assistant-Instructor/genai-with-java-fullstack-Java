프로세스란(Process) : 운영체제에서 실행 중인 각각의 프로그램(애플리케이션)을 의미


멀티태스킹이란(Multi Tasking) : 하나의 프로그램 안에서 두 개 이상의 작업을 동시에 처리하는 것을 말함. (멀티스레드로 인해 가능)
예시 1 ) 하나의 운영체체안에서 여러 개의 다른 프로그램을 동시에 실행 할 수 있는 것. 메모장작업과 영상재생 작업을 동시에 처리하는 등의 것을 말함.
예시 2 ) 카카오톡 프로그램안에서 메세지를 보내고, 받고, 동영상을 재생하고 할 수 있는 것. 


스레드란(Thread) :  프로세스 내에서 하나의 작업을 처리(실행,수행)하는 작업단위를 말함. 프로세스 내의 자원을 공유하여 사용할 수 있다. 각각의 작업단위(스레드)는 독립적으로 작동한다. 작업을 처리하는 순서가 없다.


멀티스레드란(Multi Thread) : 하나의 프로세스에서 동시에 여러 개의 작업을 수행하는 것을 말함.(프로그램 한 개의 내부안에서 일어나는 멀티태스킹을 의미)


다중 프로세스란(Multi Process) : 하나의 프로그램이 여러 개 독립적으로 사용되는 것을 의미(애플리케이션=프로그램 단위의 멀티태스킹을 말함.)
예) 크롬브라우저, 인텔리제이(IDE)등
